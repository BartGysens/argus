**Installation:

Bitbucket is an extension to the argus - a Quality control platform.

1. Unpack the argus_bitbucket folder and contents in the appropriate modules
directory of your Drupal installation.  This is probably sites/all/modules/
2. Enable the argus_bitbucket module in the administration tools.
3. If you're not using Drupal's default administrative account, make sure 
 "administer smartschool bitbucket" is enabled through access control
 administration.
4. Visit the settings page and make appropriate configurations
 at admin > config > argus > Bitbucket
