**Description:
Bitbucket module lets you integrate argus with Bitbucket for development and issue tracking.
Get your API key at Bitbucket.

**Benefits:
Devlopment as a pro.

**Installation AND Upgrades:
See the INSTALL.txt file.

**Notices:
Issues sometimes can not be sent to Bitbucket due to connection problems; please retry later.
