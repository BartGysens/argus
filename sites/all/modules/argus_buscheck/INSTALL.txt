**Installation:

Buscheck is an extension to the argus - a Quality control extension.

1. Unpack the argus_buscheck folder and contents in the appropriate modules
directory of your Drupal installation.  This is probably sites/all/modules/
2. Enable the argus_buscheck module in the administration tools.
3. If you're not using Drupal's default administrative account, make sure 
 "administer argus buscheck" is enabled through access control
 administration.
4. Visit the argus_buscheck settings page and make appropriate configurations
 at admin > argus > Buscontrole De Lijn: /admin/config/argus/buscheck
