**Installation:

Documenten generator is an extension to the argus - Smartschool platform.

1. Unpack the argus_document_generator folder and contents in the appropriate modules
directory of your Drupal installation.  This is probably sites/all/modules/
2. Enable the argus_document_generator module in the administration tools.
3. If you're not using Drupal's default administrative account, make sure 
 "administer argus document_generator" is enabled through access control
 administration.
