**Description:
Alternative tracks for registering new users as pupil or as personal.

**Benefits:
Better manipulation of large registrationforms and procedures.

**Installation AND Upgrades:
See the INSTALL.txt file.

**Notices:
In certain casses no data can be retrieved from De Lijn, the API cannot deliver
any data so no results will be shown. You can retry a bit later.
