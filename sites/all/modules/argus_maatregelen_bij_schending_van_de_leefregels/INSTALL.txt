**Installation:

Maatregelen bij schending van de leefregels is an extension to the argus - a Quality control extension.

1. Unpack the argus_maatregelen_bij_schending_van_de_leefregels folder and contents in the appropriate modules
directory of your Drupal installation.  This is probably sites/all/modules/
2. Enable the argus_maatregelen_bij_schending_van_de_leefregels module in the administration tools.
3. If you're not using Drupal's default administrative account, make sure 
 "administer argus maatregelen_bij_schending_van_de_leefregels" is enabled through access control
 administration.
