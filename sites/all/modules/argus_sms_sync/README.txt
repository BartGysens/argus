a product of the Smartschool development team.

**Description:
Smartschool Sync can collect partial datathrough the Smartschool API for use in
own projects. This service is maintained by the Smartschool development team, which
is in no way responsible for this syncing module.

**Benefits:
You can create your side platform on top of Smartschool data and in that way
independant organise your data and organisation to your own needs.

**Installation AND Upgrades:
See the INSTALL.txt file.

**Notices:
When no connections are possible to the Smartschool platform this module can not
recover any data, please be advised.
